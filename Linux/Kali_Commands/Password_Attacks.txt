# Password Attacks & Cracking Tools

- hydra -l [user] -P [wordlist] [protocol]://[ip]: Online password cracking (e.g., SSH, FTP)
- john --wordlist=[list] [hashfile]: Crack local password hashes using John the Ripper
- hashcat -m [mode] -a 0 [hashfile] [wordlist]: Advanced GPU-based hash cracking
- medusa -h [target] -u [user] -P [list] -M [protocol]: Parallel network login auditor
- ncrack: High-speed network authentication cracking tool
- crunch [min] [max] [sysbols]: Generate wordlists based on criteria
- fcrackzip: Crack ZIP archive passwords
