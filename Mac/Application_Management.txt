# macOS Application & Utility Management

- killall [ApplicationName]: Force quit an application by name
- open -a [ApplicationName]: Launch a specific application
- hdiutil mount [file.dmg]: Mount a disk image
- installer -pkg [package.pkg] -target /: Install a .pkg file from the CLI
- softwareupdate -l: List available software updates
- softwareupdate -i -a: Install all available software updates
- say "[text]": Convert text to speech via the speakers
- screenshot -c: Take a screenshot to the clipboard
- mdls [file]: List metadata attributes for a specific file (Spotlight index)
