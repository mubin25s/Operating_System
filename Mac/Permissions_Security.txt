# macOS Permissions & Security Commands

- chmod [permissions] [file]: Change file/directory permissions
- chown [user]:[group] [file]: Change file/directory ownership
- sudo [command]: Execute a command with administrative (root) privileges
- ls -ale: List files with detailed permissions and Access Control Lists (ACLs)
- csrutil status: Check System Integrity Protection (SIP) status
- spctl --master-disable: Disable Gatekeeper (not recommended for beginners)
- security: Administer certificates, keys, and keychains
- drutil: Interact with CD/DVD burning hardware and media
