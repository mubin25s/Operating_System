# macOS Homebrew (Package Manager) Commands

- brew install [package]: Install a package
- brew uninstall [package]: Uninstall a package
- brew update: Update Homebrew and formulae
- brew upgrade: Upgrade installed formulae
- brew list: List installed formulae
- brew search [text]: Search for formulae
- brew info [package]: Display information about a formula
- brew doctor: Check system for potential problems
