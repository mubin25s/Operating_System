# macOS Developer Tools & Environment

- git: Version control system (core developer tool)
- xcode-select --install: Install macOS Command Line Tools
- python3: Execute Python 3 scripts
- pip3: Package installer for Python
- node: Node.js runtime environment
- npm: Node Package Manager
- zsh: The default macOS shell (configuration at ~/.zshrc)
- nano/vim: Built-in command line text editors
- alias [name]='[command]': Create shortcuts for complex commands
- env: Display current environment variables
