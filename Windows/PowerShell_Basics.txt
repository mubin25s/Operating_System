# Windows PowerShell Basics (Modern Commands)

- Get-Command: Get all commands (cmdlets, functions, etc.)
- Get-Help: Display help for PowerShell commands
- Get-Service: Get the status of services on a computer
- Get-Process: Get the processes that are running on a computer
- Set-ExecutionPolicy: Change the user preference for the PowerShell execution policy
- Invoke-WebRequest: Get content from a web page on the internet
- Get-Content: Get the content of the item at the specified location
- Select-String: Find text in strings and files
- Start-Process: Start one or more processes on the local computer
- Stop-Process: Stop one or more running processes
- Get-History: Get a list of the commands entered during the current session
