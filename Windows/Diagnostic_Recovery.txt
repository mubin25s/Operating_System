# Windows Diagnostic & Recovery Commands

- sfc /scannow: System File Checker - scans and repairs corrupted system files
- dism /online /cleanup-image /restorehealth: Deployment Image Servicing and Management - repairs the Windows image
- gpupdate /force: Force an update of Group Policy settings
- chkdsk /f /r: Check disk and repair errors/locate bad sectors
- bootrec /fixmbr: Repair the Master Boot Record (requires recovery environment)
- taskmgr: Open Task Manager (GUI tool)
- perfmon: Open Performance Monitor
- eventvwr: Open Event Viewer to check system logs
