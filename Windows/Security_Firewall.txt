# Windows Security & Firewall Commands

- netsh advfirewall: Manage Windows Firewall settings
- sc query: Display status of services (including security services)
- icacls: Display or modify Access Control Lists (ACLs) for files and folders
- net share: Create, delete, or display shared resources
- cipher: Display or alter the encryption of directories and files on NTFS volumes
- certutil: Manage certificates and Certificate Services
- whoami /priv: Display the privileges held by the current user
- net account: Set account requirements (like password length)
